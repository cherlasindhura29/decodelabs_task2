# 🎵 BeatWave - Responsive Music Streaming Website

## 📌 Project Overview

BeatWave is a modern and responsive music streaming website inspired by popular music platforms. The project was developed to demonstrate responsive web design principles using HTML5 and CSS3 while ensuring a smooth user experience across mobile, tablet, and desktop devices.

---

## ✨ Features

* 🎨 Modern Dark Theme UI
* 📱 Fully Responsive Layout
* 🧭 Responsive Navigation Menu
* 🎵 Trending Playlists Section
* 🎤 Popular Artists Section
* 💎 Premium Features Section
* 🎼 Top Genres Section
* ⚡ Smooth Scrolling Navigation
* 🖱️ Interactive Hover Effects
* ♿ Semantic HTML5 Structure
* 📐 Mobile-First Responsive Design

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries
* Google Fonts

## 📱 Responsive Design

This project follows a Mobile-First approach and adapts seamlessly to:

* 📱 Mobile Devices
* 📲 Tablets
* 💻 Desktop Screens

CSS Media Queries are used to ensure proper spacing, alignment, and layout responsiveness across different screen sizes.

git clone : https://cherlasindhura29.github.io/beatwave-responsive-website/
